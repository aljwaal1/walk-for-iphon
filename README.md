# رفيق المشي - نسخة احترافية للآيفون

تطبيق PWA خفيف للمشي يعمل من Safari ويمكن إضافته إلى الشاشة الرئيسية.

## التشغيل على GitHub Pages
1. ارفع الملفات كما هي إلى المستودع.
2. فعّل GitHub Pages من Settings > Pages.
3. افتح الرابط من Safari في الآيفون.
4. اضغط Share ثم Add to Home Screen.
5. افتح التطبيق من الأيقونة وفعّل الإشعارات.

## OneSignal
تم دمج App ID و Safari Web ID داخل index.html.
لا تضع REST API Key داخل أي ملف عام.

## تحديث منع انطفاء الشاشة
- تمت إضافة Screen Wake Lock عند بدء جلسة المشي.
- يتم إيقاف منع القفل عند الإيقاف المؤقت أو إنهاء/تصفير الجلسة لتوفير البطارية.
- تظهر حالة واضحة داخل شاشة الجلسة: مفعّل، غير مدعوم، أو متوقف مؤقتاً.
- عند رجوع المستخدم للتطبيق أثناء الجلسة يتم طلب Wake Lock مرة أخرى تلقائياً.

## تحديث تاريخ السجل حسب الجهاز
- تاريخ السجل اليومي والأسبوعي يعتمد على تاريخ الجهاز المحلي مباشرة.
- تم منع استخدام `toISOString()` في مفاتيح الأيام لأنها تعتمد على UTC وقد تغيّر اليوم عند اختلاف المنطقة الزمنية.

## تحديثات الدقة وتجربة الاستخدام - V4
- تم تشديد فلتر GPS: ممتاز حتى 20م، مقبول حتى 35م، وأي دقة أضعف لا تُحتسب كمسافة.
- أصبحت حالة GPS أوضح داخل الشاشة: ممتاز / مقبول / ضعيف، مع تنبيه عند عدم احتساب المسافة.
- تم تخفيف المبالغة في الخطوات: عند وجود مسافة GPS يعتمد التطبيق على المسافة وطول الخطوة، والحركة تستخدم عند غياب مسافة GPS.
- تم منع حفظ جلسة قصيرة جداً أو صفرية بدون تأكيد من المستخدم.
- إذا لم تُسجل مسافة GPS، يمكن حفظ الجلسة كجلسة زمنية فقط بعد تأكيد المستخدم.
- تم إضافة تأكيد قبل تصفير جلسة نشطة حتى لا تضيع بيانات المشي بالخطأ.
- تم توضيح أن زر اختبار الإشعار هو اختبار محلي وليس إشعار OneSignal من السيرفر.
