# رفيق المشي - PWA للآيفون

تطبيق مشي عربي يعمل عبر GitHub Pages ويمكن إضافته إلى شاشة الآيفون.

## الملفات
- index.html
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png

## طريقة التشغيل على GitHub Pages
1. أنشئ مستودع جديد على GitHub.
2. ارفع كل الملفات في جذر المستودع.
3. افتح Settings > Pages.
4. اختر Branch: main ثم Save.
5. افتح الرابط الناتج من Safari على iPhone.
6. اضغط زر المشاركة ثم Add to Home Screen / إضافة إلى الشاشة الرئيسية.

## ملاحظات مهمة
- GPS يعمل عندما التطبيق مفتوح.
- iOS قد يوقف GPS عند إغلاق الشاشة، لذلك أضفنا التسجيل السريع.
- تنبيهات الشاشة المغلقة تحتاج Web Push فعلي عبر خادم أو خدمة خارجية، وهذه النسخة جاهزة لاستقبال Push لاحقاً من خلال service-worker.js.
